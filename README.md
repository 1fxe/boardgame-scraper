### Board Game Scraper

Web scraper for [boardgamegeek.com](https://boardgamegeek.com/)

## Usage

```go
-getCategories
    Gets list of categories for future parsing
-getMechanics
    Gets list of mechanism for future parsing
-parseCategories
    Parse categories from list
-parseMechanics
    Parse mechanisms from list
```
